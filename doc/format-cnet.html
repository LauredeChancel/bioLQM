CNET format

Format used by BNS, a tool for the identification of synchronous attractors. Models are represented as a list of partial truth tables, similar to the Berkeley Logic Interchange Format. 

.v 3
# component order:
# [A, B, D]

# A
.n 1 1 1
0 0
1 1

# B
.n 2 1 1
0 1
1 0

# D
.n 3 2 1 2
0- 0
10 0
11 1
