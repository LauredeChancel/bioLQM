Pint format

The format used by the Pint static analysis tool.

"A" [0, 1]
"B" [0, 1]
"C" [0, 1]


"B" 0 -> 1 when "A"=0
"B" 1 -> 0 when "A"=1

"C" 1 -> 0 when "A"=0
"C" 0 -> 1 when "A"=1
