MaBoSS format

Format used by the MaBoSS tool. This format generates a .bnd file with the model itself, and a .cfg file to define parameter values. 

Node A {
  logic = (A);
  rate_up = @logic ? $u_A : 0;
  rate_down = @logic ? 0 : $d_A;
}

Node B {
  logic = (!A);
  rate_up = @logic ? $u_B : 0;
  rate_down = @logic ? 0 : $d_B;
}

Node C {
  logic = (A);
  rate_up = @logic ? $u_C : 0;
  rate_down = @logic ? 0 : $d_C;
}
